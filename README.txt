INSTRUCCIONES

1. Abrir VS Code.
2. Crear una carpeta para el proyecto.
3. Copiar app.py y requirements.txt en esa carpeta.
4. Abrir la terminal en VS Code.
5. Ejecutar:

   pip install -r requirements.txt

6. Luego ejecutar:

   streamlit run app.py

7. Se abrirá la app en el navegador.
8. Presionar el botón: "Actualizar datos y entrenar modelo".

La app descarga datos reales de EIA, entrena el modelo, muestra métricas, genera la predicción del siguiente periodo y guarda un histórico en CSV.
