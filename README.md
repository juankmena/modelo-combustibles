# Modelo de predicción de precios de combustibles

Aplicación desarrollada en Python utilizando Streamlit para estimar la probabilidad de incremento en precios de gasolina con datos de la EIA.

## Funcionalidades

- Descarga automática de datos EIA
- Entrenamiento de modelo (Decision Tree)
- Validación cruzada
- Predicción del siguiente periodo
- Interfaz gráfica interactiva

## Ejecución

```bash
pip install -r requirements.txt
streamlit run app.py
